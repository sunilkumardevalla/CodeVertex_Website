# 90-Day Enterprise Roadmap

Date: February 19, 2026

## Objective
Move from launch-ready website to enterprise revenue engine with security, compliance, and growth loops.

## North-star KPIs (first 90 days)
- MQL to SQL conversion rate >= 18%
- Contact form to booked call rate >= 12%
- Median first response SLA <= 4 business hours
- Organic sessions +30%
- Security incidents affecting website: 0

## Phase 1 (Days 1-30): Foundation
1. Domain and DNS hardening
- Enforce apex + www strategy, HSTS preload readiness, TLS monitoring.
2. Observability and alerting
- Add uptime monitor, error tracking, and Slack/email on-call routing.
3. Analytics governance
- GA4 + server-side event map + attribution naming standard.
4. HubSpot operations baseline
- Define lifecycle stages, lead status mapping, ownership, and SLA tasks.

## Phase 2 (Days 31-60): Conversion and Trust
1. Content engine
- Publish 8 high-intent blog posts and 2 downloadable lead magnets.
2. CRO program
- Run 3 A/B tests: hero messaging, CTA hierarchy, form friction.
3. Compliance completion
- Final privacy/cookie language, consent evidence retention, CMP policy.
4. Email deliverability
- SPF, DKIM, DMARC policy to quarantine, mailbox monitoring.

## Phase 3 (Days 61-90): Scale and Reliability
1. Revenue ops automation
- Automatic task routing, deal aging alerts, no-response escalations.
2. Security operations cadence
- Monthly dependency scan review + quarterly external test planning.
3. BCP and incident drills
- Backup/restore validation + tabletop exercise with comms template.
4. Executive reporting
- Weekly KPI snapshot: traffic, conversion, pipeline, close rate, risk.

## Owners
- Growth: Marketing lead
- Revenue Ops: Sales ops / founder
- Security: Security owner
- Engineering: Web engineering owner
- Compliance: Legal/compliance owner

## Weekly cadence
- Monday: KPI review + sprint priorities
- Wednesday: execution checkpoint
- Friday: risk review + next-week scope
