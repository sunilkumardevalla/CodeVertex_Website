- Cookie categories: essential, analytics, marketing
- Consent banner with explicit choice and persistence

- Public cookie policy page
- Data retention matrix
- Subprocessor list with update date
- DPA workflow for enterprise deals

## Audit readiness
- Quarterly consent audit sample
- Evidence package stored with timestamps
