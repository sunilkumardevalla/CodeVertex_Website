Workshop Agenda

Strengthening Families: session outline and facilitation team

A clear week-by-week structure designed to translate workshop learning into home implementation.

Program agenda

Each session builds on the previous one, moving from communication foundations toward implementation and sustainability. Families receive practical exercises between sessions.

  • Session 1: communication foundations and listening frameworks
  • Session 2: boundaries, expectations, and accountability systems
  • Session 3: conflict repair and emotional regulation tools
  • Session 4: long-term family implementation plan

Facilitator team

Programs are led by licensed clinicians and family educators with workshop delivery experience. This blended facilitation model combines clinical depth with practical implementation support.

  • Clinical lead: treatment-informed family systems guidance
  • Family educator: practical exercise facilitation and coaching
  • Support team: implementation follow-up and resources
